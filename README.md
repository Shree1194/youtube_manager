# 📺 YouTube Video Manager (Python CLI)
A simple command-line based Python application to manage YouTube video details.
Supports basic CRUD operations with persistent JSON storage.
## 🚀 Features
- List all YouTube videos
- Add new video details
- Update existing video information
- Delete videos
- Data stored locally using JSON
- Clean CLI menu using Python `match-case`
## 🛠 Tech Stack
- Python 3.10+
- JSON (built-in module)
- File Handling
